# Markdown Syntax
Markdown syntax that can be used anywhere in Github or `.txt` files as well.

## Headers
```
  # Heading1 ------------------>    <h1>
  ## Heading2 ----------------->    <h2>
  ### Heading3 ---------------->    <h3>
  .
  .
  ###### Heading6 ------------->    <h6>
```  

## Emphasis
```
  *Italic Text*
  _Italic Text_
  
  **Bold Text**
  __Bold Text__
```
